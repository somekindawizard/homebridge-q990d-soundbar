Q990D Soundbar — OAuth Setup

This wizard authenticates the plugin with SmartThings. You'll need a SmartThings OAuth app created via the SmartThings CLI first.

On macOS, install the CLI with: brew install smartthingscommunity/smartthings/smartthings
Then create an OAuth app: smartthings apps:create
Use https://httpbin.org/get for both Target URL and Redirect URI.
See INSTALL.md for detailed instructions.
